Put together by Danny Boyd

  • Map geometry data comes from The Humanitarian Data Exchange
  • Displaced peoples data comes from the International Organization for Migration's Global Displacement Tracking Matrix via HumData. I've used the 25th round of surveys here.

This project uses several 3rd-party libraries

  • Bluebird.js for async promises
  • D3 Legend by Susie Lu for the pretty legend
  • d3-scale-chromatic for the color interpolation. It's a standard in v5 but not v4, so it's included in a script tag at the top
